Container als Tool

Maven

Without Docker 


mvn archetype:generate -DgroupId=nl.ovs.demo \
  -DartifactId=demoapp \
  -DarchetypeArtifactId=maven-archetype-quickstart \
  -DinteractiveMode=false

Maven

Dockerized 


docker run -it --rm --name my-maven-project \
  -v "$PWD":/usr/src/mymaven \
  -w /usr/src/mymaven \
  maven:3.2-jdk-7 \
  mvn archetype:generate -DgroupId=nl.ovs.demo \
  -DartifactId=demoapp \
  -DarchetypeArtifactId=maven-archetype-quickstart \
  -DinteractiveMode=false

Maven

Install 


docker run -it --rm --name my-maven-project \
	-v "$PWD":/usr/src/mymaven \
	-w /usr/src/mymaven \
	maven:3.2-jdk-7 \
	mvn clean install
